3. Define “referência relativa” e “referência absoluta”. Dá um exemplo de cada. (3 valores)

        Resposta:referência relativa ela muda automaticamente quando a fórmula é copiada para outra célulua.
   ex: =A1+B1

   referência absoluta ela nao muda,mesmo quando a fórmula é copiada.
   ex:=$B$2*C2
   =$E$3/2
   =$E$3/2
